## Features

-   **Task Management**: Easily add, edit, and delete tasks.
-   **Focus Timer**: Track the time spent on each individual task with a built-in timer.
-   **Daily Goals**: Set your daily task limit (1-10 tasks) using an interactive slider.
-   **Progress Tracking**: Monitor your daily progress with a visual completion counter and total focus time summary.
-   **Dark/Light Mode**: Toggle between a crisp light theme and a sleek dark mode for comfortable viewing in any environment.
-   **Responsive Design**: Fully responsive interface that works seamlessly on desktop and mobile devices.
-   **Interactive UI**: Smooth animations and transitions for a premium user experience.

## How it Works

1.  **Set Your Goal**: Use the slider to determine how many tasks you want to tackle today.
2.  **Add Tasks**: Enter your task descriptions in the cards.
3.  **Start Tracking**: Click the "Start" button on a task to begin the timer.
4.  **Pause/Resume**: Need a break? Pause the timer and resume when you're ready.
5.  **Complete Tasks**: Mark tasks as done when finished.
6.  **Redo Tasks**: If you need to redo a completed task, use the "Redo Task" button to mark it as incomplete.

## Tech Stack

-   **React**: Frontend library for building the user interface.
-   **TypeScript**: Adds static typing for better code quality and developer experience.
-   **Vite**: Fast build tool and development server.
-   **Tailwind CSS**: Utility-first CSS framework for styling.
-   **Lucide React**: Beautiful and consistent icons.
